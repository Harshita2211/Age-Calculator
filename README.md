
#  Age Calculator

A simple and responsive **Age Calculator** built using **HTML, CSS, and JavaScript**.  
This project allows users to enter their date of birth and instantly calculates their **age in years, months, and days**.

---

##  Features
-  Input date of birth  
-  Calculates accurate age (years, months, days)  
-  Fully responsive design  
-  Clean and modern UI with CSS  
-  Instant calculation using JavaScript  

---

##  Tech Stack
- **HTML5** – Structure  
- **CSS3** – Styling & layout  
- **JavaScript (ES6)** – Logic & interactivity
